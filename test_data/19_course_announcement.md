## Course Description

Explore cutting-edge neural network approaches to NLP. We'll cover word vectors, transformers, large language models, and practical applications. Expect significant programming (Python/PyTorch) and a substantial research project.

## Prerequisites

- CS 229 (Machine Learning) or equivalent
- Strong Python programming skills
- Linear algebra and probability
- Prior deep learning experience helpful but not required

## Course Components

**Lectures (30%):**
- Neural language models
- Attention mechanisms
- BERT, GPT, and modern LLMs
- Efficient training and inference
- Ethics and bias in NLP

**Assignments (40%):**
- 5 programming assignments (PyTorch)
- Implement transformer from scratch
- Fine-tune pre-trained models
- Build QA system

**Final Project (30%):**
- Teams of 3-4 students
- Original research or substantial application
- Paper (8 pages) + presentation
- Due: March 15, 2025
